5. Sockets

Warning: Do not name a python file `socket.py`, otherwise python will get confuzzled!
```python
#!/bin/python3

import socket

HOST = '127.0.0.1'
PORT = 7777

s = socket.socket(socket.AF_INET, socket.SOCK_STREAM) #Stores ipv4 (AF INET) and port connecting (SOCK STREAM)

s.connect((HOST,PORT))
```
